# Wiki Read Page Tool

Read the full content of a specific wiki page.

## Usage

Provide a path in the format `category/name`:
- `sources/my-document` — Read a source summary
- `entities/john-smith` — Read an entity page
- `concepts/machine-learning` — Read a concept page
- `synthesis/comparison-analysis` — Read a synthesis page

## When to Use

- After `wiki_query` to get the full page content
- When you know the exact page name from the wiki index
- To follow cross-references between wiki pages
